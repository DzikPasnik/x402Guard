  <action>
1. **src/retry.ts**: Implement `fetchWithRetry(url, init, config)`:
   - Config: `maxRetries` (default 3), `retryBaseMs` (default 1000), `logger` (pino instance)
   - On 429: read `Retry-After` header (seconds), wait `retryAfter * 1000 * 2^attempt` ms, retry
   - On network error (TypeError from fetch): retry with exponential backoff
   - On all other errors: throw immediately (no retry for 4xx/5xx except 429)
   - Log retry attempts at WARN level via pino
   - Return Response on success or throw after max retries exhausted
   - Export RetryConfig interface

2. **src/client.ts**: Implement `X402GuardClient`:
   - Constructor takes `X402GuardConfig`, validates with zod schema from types.ts. Falls back to env vars: `process.env.X402GUARD_PROXY_URL`, `process.env.X402GUARD_AGENT_ID`
   - Config is frozen after construction (immutable -- project coding style rule)
   - All methods return typed responses, never raw Response objects
   - Methods (all async):
     - `healthCheck(): Promise<boolean>` -- GET /health, returns true if 200, throws ProxyUnreachableError with message "Proxy unreachable at {url} -- is docker compose up running?" on failure
     - `createAgent(req: CreateAgentRequest): Promise<Agent>` -- POST /agents
     - `getAgent(id: string): Promise<Agent>` -- GET /agents/:id
     - `createRule(agentId: string, rule: CreateRuleRequest): Promise<GuardrailRule>` -- POST /agents/:agentId/rules, body is `{ rule_type: rule }` (note: the proxy expects `rule_type` field containing the RuleType discriminated union)
     - `listRules(agentId: string): Promise<GuardrailRule[]>` -- GET /agents/:agentId/rules
     - `createSessionKey(agentId: string, req: CreateSessionKeyRequest): Promise<SessionKey>` -- POST /agents/:agentId/session-keys
     - `revokeAll(agentId: string, req: RevokeAllRequest): Promise<void>` -- POST /agents/:agentId/revoke-all
     - `proxyPayment(req: ProxyRequest): Promise<ProxyResponse>` -- POST /proxy
     - `proxySolanaPayment(req: SolanaProxyRequest): Promise<SolanaProxyResponse>` -- POST /proxy/solana
   - All methods: parse ApiResponse envelope, throw GuardrailViolationError on 403, throw X402GuardError on other non-2xx
   - Private `request<T>(method, path, body?): Promise<T>` handles JSON serialization, content-type header, retry delegation, response parsing

3. **src/index.ts**: Add re-exports for client.ts and retry.ts.
